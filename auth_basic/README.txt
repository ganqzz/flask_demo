# Flask Authentication and Authorization without any Flask auth extensions(ex. Flask-Login)

- Blueprint
- SQLAlchemy
- WTForms
- Mail
- Celery

```
flask run
```

- Authentication
    * Registration
    * Login / Logout by using Sessions and Cookies
    * Remember me / Forget me

- Authorization
    * Roles

- Activation
    Registration -> Confirmation email -> Activation

- Reset password
    Send reset password email -> Set new password -> Password reset
